## 🔍 Quick Facts Reference

### Cloud Coins Discount
- **Conversion:** 2000 coins = ₹100 discount
- **Ratio:** 20 coins = 1 rupee
- **Minimum:** 20 coins for ₹1 discount
- **Location:** `public/upgrade-form.html`
- **API:** `/api/coins/balance`

### Storage Card
- **Location:** `public/upload.html` (after drag zone)
- **Updates:** Real-time from `/storage/{email}`
- **Circle:** SVG with smooth animation
- **Icon:** 💾 spinning 4 seconds
- **Displays:** Used / Total / Available storage
